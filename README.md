# RegresionPolinomial
## Actividades Realizadas 
Para el conjunto de datos asociado a perfiles de gris se ajustará un polinomio de grado 3 mediante el método de Vandermonde 
1. Tomando el primer renglón de cada archivo como las coordenadas y de un conjunto de datos (x, y), ajuste un polinomio de grado 3 a cada primer renglón de los 10 archivos.
2. Promedie los datos de tal forma que pueda obtener un prolinomio promedio derivado de los primeros renglones de los 10 archivos.
3. Calcule el error cuadrático asociado al polinomio ajustado en cada caso.
4. Se deberán tener 11 polinomios de grado 3. 1 de cada primer renglón de un archivo ”.txt”, siendo 10 en total; y uno más obtenido del promedio de los datos.
